## About JUnit Commons

Common extensions of JUnit for supporting the wcm.io JUnit tests.

### Maven Dependency

```xml
<dependency>
  <groupId>io.wcm</groupId>
  <artifactId>io.wcm.testing.junit-commons</artifactId>
  <version>1.0.0</version>
</dependency>
```
